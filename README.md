# IBENO — Boutique en ligne

Site e-commerce de vêtements et accessoires pour la Côte d'Ivoire.

## Paiements acceptés
- 🌊 Wave
- 🟠 Orange Money  
- 🔵 Moov Money
- 🟡 MTN MoMo
- 💳 Visa / CB

## Lancer en local
```bash
npm install
npm run dev
```

## Déploiement
Ce projet est déployé automatiquement sur Vercel.
